BagMan
------

A file-based collection manager.

Operations
----------

1. Create Collection (***)
 - Select files, folders.
 - Choose Collection Type.
 - Choose Destination (in place, known location, etc).
 - Create...
 - Collect Metadata.
 - Record Result?
 - Finish.

2. Clone Collection (**)
 - Choose Collection(s).
 - Choose Destination.
 - (Choose new Collection Type)
 - Clone.
 - Record Result?
 - Finish.
 
3. Validate Collection (***)
 - Choose Collection(s).
 - Validate.
 - (Review Changes?)
 - Record Result?
 - Finish.

4. Analyse Collection (**)
 - Choose Collection(s).
 - Choose Analysis Tools (DROID etc).
 - Analyse.
 - Review/Export Results.
 - Finish.

5. Modify Collection (**)
 - Choose Collection(s).
 - Choose Operation: Change Collection Type, (Merge/Split)
 - Modify.
 - Record Result?
 - Finish.

6. Send Collection (*)
 - Choose Collection(s)
 - Choose Remote Destination...
 
7. Receive Collection (*)
 - Choose source (URL? Bag? Email? What?)
 - Choose Destination.

Notes
-----

All 'Choose Collection(s)' require selection from list of known collections.

Priority is indicated by number of stars.
